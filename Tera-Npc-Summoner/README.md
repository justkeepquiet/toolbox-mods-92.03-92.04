# Npc-summoner

## _Открытие NPC / "торговца" с любого места. _
------
# Использование

Комманда | Описание
| ------------- | ------------- | 
| `sum` / `s` | Открыть окно UI с разными NPC для призыва |

Чтобы обновить индификаторы NPC телепортруйтесь в ***Верхний Дозор*** и ID обновятся автоматически>
